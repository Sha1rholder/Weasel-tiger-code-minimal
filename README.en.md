# Minimalist Tiger Code Weasel Configuration

- Modified from虎码秃版, kept Tigress (Word Input Version of Tiger Code) only
- Use `~` prefix for Pinyin reverse lookup
- Removed all other shortcuts and prefix keys
- Retained only Simplified Chinese characters
- Single quotes output `'` directly (can be changed back to Chinese single quotes `‘’` in [symbols.yaml](symbols.yaml))
